Resources

A list of software/tools suggested for sfhacks. Create accounts for the following websites and download software as necessary.

  • Atom - One recommended text editor, especially for web development. Completely free and open source. Download various packages and themes to your own liking as well. We suggest Minimap, Color Picker, and the Material Themes to start.
  • Devpost - Used during hackathons to submit projects
  • Eclipse - The IDE we use for Java programming competitions.
  • IntelliJ - An alternative Java IDE.
  • GitHub - Make your own GitHub account here. GitHub will be heavily used to manage your projects through repositories
  • GitHub Desktop - A simpler way to commit to your own repositories.
  • HackerRank - For mock competitions that we will be doing in meetings, sign up for HackerRank through GitHub.
  • Java SE 8 - In order to program in Java, you first need to download it. Follow the instructions on the Oracle website.
  • Python - A good language to learn for starting programming, Python is now also being used by our members to work with complex projects in Artificial Intelligence. Follow the instructions on the Python website.
  • NodeJS - In order to use NodeJS, you need to download it. Follow the instructions on the NodeJS website, and you're all set.

  • Swift Playgrounds - An application for iOS 10 to help learn Swift. Playgrounds is approved so it can be downloaded on our school iPads.
